# DatePicker日历输入组件

本组件依赖my97datepicker组件


## 快速使用

```````````````html
<span data-role="wejs-date-picker"
value="now"
name="date"
min="now"
format="yyyy-MM-dd HH:mm:ss"
></span>
```````````````



## 属性配置
------------------------------
### min
* 说明：最小日期
* 类型：字符串,例如：2015-01-01
* 默认值：无
